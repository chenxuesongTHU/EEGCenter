# EEGCenter

## 目的

为方便**THUIR BMI search**组的同学进行快速脑电分析、脑电相关模型构建和新同学的上手，
和相关人员的学术和应用研究，
特此维护**EEGCenter**。

## 功能介绍

**EEGCenter**以**mne**包为基础，提供不同格式脑电数据的统一读取接口，
并直接提取计算机和医学领域常用的脑电分析特征，
用于不同baseline模型和sota模型的训练和统一的模型指标计算。
